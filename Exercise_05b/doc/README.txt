Exercise 5b: AssignTwoType
~~~~~~~~~~~~~~~~~~~~~~~~~~

In this exercise you will explore a different way to structure your code within different packages. The code from AssignTwoType is now split over 4 packages. Otherwise the exercise is the same.

To run the calculator type \code{run} when building the project (or \code{make run} from a terminal).

To run this exercise type [omc run.mos] or just execute run.bat.
If you need to edit the input of the calculator edit the file called input.txt.
